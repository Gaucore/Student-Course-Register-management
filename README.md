# 🎓 Student Course Registration System

A console-based Java application that manages students, courses, enrollments, fee status, and reports using Core Java concepts.

---

# 📖 Project Overview

The Student Course Registration System is a menu-driven Java application developed using Object-Oriented Programming (OOP) and the Java Collections Framework.

It allows an institute to manage students, courses, enrollments, fee payments, sorting, and reporting through a simple console interface.

This project demonstrates how multiple Core Java concepts work together in a real-world application.
